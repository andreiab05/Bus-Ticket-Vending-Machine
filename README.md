# Bus-Ticket-Vending-Machine
## Overview

This C++ application implements a layered architecture for a bus ticket vending machine system. The program allows users to purchase tickets or subscriptions, receive change, and manages the machine's internal currency. It also supports CRUD operations for tickets and subscriptions.

## Technologies
- C++
- Object-Oriented Programming (OOP)
- Layered Architecture

## Features
- Buy bus tickets
- Purchase subscriptions
- Manage machine currency and change
- CRUD operations for tickets and subscriptions
